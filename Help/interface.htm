Интерфейс программы

📊 Главное окно

Главное окно программы разделено на несколько областей:

  • Меню - доступ ко всем функциям программы
  • Панель инструментов (ToolBar) - быстрый доступ к основным командам
  • Параметры задачи - ввод начальных условий и функции
  • Таблица результатов - отображение значений x, y, f(x,y)
  • График решения - визуализация полученного решения
  • Статусная строка - отображение текущего состояния
