Модифицированный метод Эйлера

Математическая постановка задачи

Решается задача Коши для обыкновенного дифференциального уравнения первого порядка:

dy/dx = f(x, y), y(x0) = y0

на отрезке [x0, X] с шагом h.

Формула метода

Модифицированный метод Эйлера (метод Эйлера-Коши) имеет второй порядок точности:

y_new = y + h · f(x + h/2, y + (h/2) · f(x, y))

Алгоритм:

1. Задать начальные значения x0, y0
2. Вычислить количество шагов: n = (X - x0) / h
3. Для i = 0 до n-1:
   x = x0 + i·h
   y_new = y + h·f(x + h/2, y + (h/2)·f(x, y))
   x = x + h
   y = y_new
   Запомнить (x, y)
4. Построить график полученных точек
