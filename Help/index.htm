Главная Интерфейс Алгоритм Интеграция Экспорт

Решение дифференциальных уравнений

Модифицированный метод Эйлера

О программе

Программа предназначена для численного решения обыкновенных дифференциальных уравнений первого порядка вида dy/dx = f(x, y) с использованием модифицированного метода Эйлера (метода Эйлера-Коши).

Основные возможности

  • Решение ДУ модифицированным методом Эйлера
  • Визуализация решения в виде 2D графика (OxyPlot)
  • Таблица результатов (x, y, f(x,y))
  • Автоматический пересчёт при изменении параметров
  • Экспорт в Excel (таблица + график на одном листе)
  • Экспорт в Word (полный отчёт с графиком)
  • Интеграция с MATLAB (вызов ode45, 3D графики)
  • Интеграция с Mathcad Prime (передача параметров)
  • Сохранение графика в PNG
  • Автоматическое сохранение/загрузка параметров (INI)
  • MVVM архитектура
  • Асинхронный вызов MATLAB
